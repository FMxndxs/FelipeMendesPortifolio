About Me

Formando em Ciência da Computação pela Universidade Presbiteriana Mackenzie, com experiência em desenvolvimento front-end e back-end (C, C++, CSS, HTML, JavaScript, Python e Java). Estudo estruturas de dados, bancos de dados e orientação a objetos. Tenho vivência internacional e ótima conversação e leitura em espanhol e inglês. Busco oportunidades para aplicar minhas habilidades em programação, análise de sistemas e desenvolvimento web.

My Projects

Sistema Hotel

Sistema de cadastro e gerenciamento de hóspedes em um hotel. Permite cadastro de hóspedes, reservas e controle de quartos. Desenvolvido em C++ com interface gráfica.

Loja de Roupa (Base)

Projeto de site desenvolvido para uma loja de roupas focada em estilo e identidade. A proposta visual combina design moderno e navegação intuitiva, destacando produtos com alta qualidade visual e responsividade total. Experiência otimizada para conversão e engajamento.

Portal de Alunos

Sistema de cadastro de alunos e gestão de matérias e notas. Permite o cadastro de alunos, de matérias, ra, entre outros, além de controle de notas, acesso as listas e a remoção das mesmas. Desenvolvida com linguagem java, orientação objetos e estruturas de herança (Extends).

Sistema de Academia

Sistema de cadastro e gerenciamento de treinos para academia. Permite o registro de exercícios, cálculo de calorias, definição de metas semanais e relatórios de desempenho. Desenvolvido em Python com interface de console.

Jogo Dominó

Jogo simples de dominó com regras básicas e partidas para dois jogadores. Permite distribuição aleatória das peças, verificação de jogadas válidas e controle de turnos. Desenvolvido em C com interface de terminal.

Jogo da Vida (Conway)

Simulação baseada no autômato celular criado por John Conway. O sistema representa uma grade de células que evoluem em gerações conforme regras simples de nascimento, sobrevivência e morte, baseadas no número de vizinhos vivos.
